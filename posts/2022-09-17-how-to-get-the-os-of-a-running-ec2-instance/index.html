How to Get the OS of a Running EC2 Instance

2022-09-17
0 minute read
aws , EC2

Go to the EC2 console and the select the instance what you want to check. Then go to Action -> Instance setting -> Get instance screenshot

In the screenshot you can find the current version of OS.
